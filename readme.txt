== Description ==

if you have multiple contact on a website but don’t want to expose them all at a time, that’s the plugin do. 
It will only show a contact of your choice or your visitors choice. Contact is changeable through a simple URL.

for demo: http://demo.pupungbp.com/contact-referal

== Installation ==

1. Upload `contact-referal` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use widget or either shortcode to display the contact in your desired page.

== Frequently Asked Questions ==

= How do I display the contact information =
You can use 'Contact Referal' widget on Appearance > Widgets. Or you can simply use [c_referal] shortcode and place it in any page.

= Do this plugin has any other option =
Currently the widget can choose between full display, or simple display. If you use shortocde use [c_referal style="simple"] to display simple layout, or [c_referal style="full"] for full layout, or just simply use [c_referal] for default settings.
